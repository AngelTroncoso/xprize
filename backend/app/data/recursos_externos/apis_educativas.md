# APIs Educativas y Recursos Externos

## Khan Academy
- palabras clave: ejercicios, series numéricas, suma, resta, operación, fracciones
- recurso: ejercicios interactivos de matemáticas para 3° básico
- uso recomendado: práctica guiada de conceptos aritméticos, descomposición y resolución paso a paso.

## GeoGebra
- palabras clave: recta numérica, representación geométrica, visualización, fracciones, número mixto
- recurso: construcciones dinámicas de recta numérica y manipulación visual de fracciones
- uso recomendado: representaciones concretas de cantidades y comparación de fracciones.

## PhET Simulaciones
- palabras clave: simulación, experiencia interactiva, visualización, comprensión, modelo concreto
- recurso: simulaciones de física y matemáticas para entender patrones y relaciones numéricas
- uso recomendado: aprendizaje experimental con material visual que apoya la comprensión de `parte de un todo` y `comparación de fracciones`.

## Mapeo Conceptual
- `fracciones`: Khan Academy, GeoGebra, PhET Simulaciones
- `serie numérica`: Khan Academy, GeoGebra
- `representación`: GeoGebra, PhET Simulaciones
- `comparación de fracciones`: Khan Academy, GeoGebra
- `recta numérica`: GeoGebra
